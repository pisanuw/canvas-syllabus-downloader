Course Description

Examines the fundamental issues in designing and developing computer video games; creative and artistic elements, story narration, software architecture, interaction model, mathematic, physics, special effects, and in-game AI logic. Experiences elements in game design: world setting, game play, and interface; and experiences implementing games: conceptualization, prototyping, and play testing. Prerequisite: STMATH 125 or MATH 125; and CSS 340 or CSS 342. 

Learning Objectives

  • Critically examine video games
  • Understand the structure of games
  • Design, prototype, test and implement a game from scratch
  • Understand and extend techniques commonly used in games
  • Work in groups, present and reflect on extended project

Textbooks

Grading

Required

  • Final group project (0.5 points)

Optional

  • Up to 5 individual projects (0.5 points each, max 2.5)
  • 2-person project (0.5 points)
  • Game mechanic writeup with code and video (0.5 points)

To get 2.0, you can

  • Complete the Final group project and 3 individual projects, OR
  • Complete the Final group project, 1 individual project, the 2-person project and the game mechanic writeup

To get 4.0, you need to

  • Complete the Final group project, 5 individual projects, the 2-person project and the game mechanic writeup

Rules for Deliverables

  • Each deliverable is marked as accepted/not accepted. If it is not accepted, you have one week to revise and resubmit it. If it gets "not accepted" a second time, you cannot count that project towards the course grade.
  • If your project is very similar to a previous project and does not demonstrate learning new concepts, it will be graded as "not accepted"
  • The "Final" has to be delivered at the end of Week-10.
    • The final has additional sub-deliverables: digital prototype-1 (week 6), digital prototype-2 (week 7), alpha testing (week 8) and beta testing (week 9)
  • You cannot deliver more than one project per week
    • If you are going to submit 6 projects, the 2-person project and the game mechanic writeup that means you have to essentially turn in a new project each week
    • If you are doing 3 projects, you can turn them in as late as week-7, week-8 and week-9
  • All deliverables are due by Sunday midnight via Canvas.
