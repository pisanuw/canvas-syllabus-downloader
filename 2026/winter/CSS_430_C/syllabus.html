Course Description

Dive deep into the architecture of modern operating systems with this comprehensive course. Using C/C++ and Java, we will explore fundamental OS concepts like processes, threads, and memory management. We will also investigate advanced topics such as synchronization, deadlocks, and virtual memory. To stay current with industry trends, we'll utilize cloud compute virtualization, software repositories and generative AI tools, providing a lite introductory foundation for modern software development as we explore the programming assignments.

Prerequisites: CSS 343

Course Goals

The overall goal of CSS 430 is to learn fundamental concepts that are used in and applicable to a variety of operating systems. The course consists of three major concepts:

  1. Process Management: Scheduling, executing, synchronizing, and terminating application programs.
  2. Memory Management: Loading programs and dynamically allocating/deallocating memory space.
  3. File Systems: Mechanisms for storage and access to data and programs residing on disks.

Through the course, we will use Java to illustrate many operating-system concepts. As with most technical courses, expect to spend an additional 10 to 15 hours a week outside of class time including reading, independent research, and coding.
